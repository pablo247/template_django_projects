## How to use

Create root folder of the project
```sh
mkdir folder_project
```
Clone this project in folder root
Move template files
```sh
mv template_django_projects/template/* .
mv template_django_projects/template/.* .
```

Configure .env file

Run bash script
```sh
chmod +x init.sh; ./init.sh
```
